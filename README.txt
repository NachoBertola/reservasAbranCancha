Sitio de Reservas Abran Cancha
Bienvenido al repositorio del proyecto "Sitio de Reservas Abran Cancha". Esta aplicación web permite a los usuarios reservar turnos de fútbol 5 y del salón de eventos en un complejo deportivo. Además, proporciona herramientas de administración para los gerentes, quienes podrán gestionar las reservas realizadas.

Características Principales
Reserva de turnos de fútbol 5: Los usuarios pueden reservar turnos para jugar en las canchas de fútbol 5 disponibles en el complejo.

Reserva del salón de eventos: Los usuarios pueden realizar reservas para eventos especiales en el salón de eventos del complejo.

Administración de reservas: Los gerentes tienen acceso a un panel de administración que les permite gestionar las reservas, incluyendo la capacidad de eliminar turnos y visualizar las reservas programadas.

Tecnologías Utilizadas
El proyecto ha sido desarrollado utilizando las siguientes tecnologías:

HTML: para estructurar la interfaz y el contenido de la aplicación.

CSS: para el diseño y la presentación visual de la aplicación.

JavaScript: para la implementación de la lógica y la interactividad de la aplicación.

Node.js: como entorno de ejecución del servidor para la administración de las reservas.

Firebase:

Firebase Authentication: para proporcionar servicios de autenticación y control de acceso de usuarios.
Firebase Firestore: como base de datos NoSQL para almacenar y gestionar la información de las reservas.
Firebase Hosting: para alojar la aplicación web y permitir su acceso desde cualquier lugar.

Instrucciones de Instalación
No se requiere ninguna instalación adicional para utilizar la aplicación, ya que está disponible en línea y alojada en Firebase Hosting.

Configuración del Proyecto
Luego de descargar el repositorio, es necesario realizar la configuración de los servicios de Firebase. Para esto, en el archivo firebaseConfigs.js se muestra una plantilla que se debe completar con las credenciales del proyecto. Además, se debe renombrar ese archivo a firebaseConfig.js para que funcionen las importaciones en los documentos HTML.

Estado del Proyecto
El proyecto se considera una versión estable (v1.0.0), lo que significa que todas las funcionalidades principales están implementadas y listas para su uso. La única funcionalidad que se encuentra en desarrollo es la de enviar notificaciones por correo electrónico, que se espera implementar en futuras actualizaciones.
